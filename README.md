## Watchlist

Watchlist is a Stock tracker Android application in which I test certain Android development techniques.

## Architecture

- Single Activity
- MVVM

## Tools

- RxJava2
- Dagger2
- Retrofit2
- Room
- LiveData
- Navigation
- DataBinding
- BindingAdapter
- JUnit
- Mockito
